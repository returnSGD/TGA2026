# 《猫语心声》美术资产需求表

> **项目**：猫语心声 | **引擎**：Unity 6 + TEngine | **美术风格**：2D手绘暗调治愈风（参考锈湖/纸嫁衣） | **场景架构**：移步换景式房间制 | **开发周期**：16周（4个月Demo版）

---

## 一、总体美术风格规范

### 1.1 视觉风格定位

| 维度 | 说明 |
|------|------|
| **核心风格** | 2D手绘暗调治愈风——沉静暗色背景 + 暖黄灯光局部照亮，形成冷暖对比 |
| **参考作品** | 锈湖系列(Rusty Lake)：沉静手绘质感、暗调室内场景、固定镜头构图；纸嫁衣系列：东方民俗手绘、暗调中的温暖色调点缀 |
| **质感** | 手工笔触纹理、旧纸张质感、轻微做旧感、不规则手绘线条（非均匀粗细） |
| **光影** | 暗调环境 + 局部暖光聚焦（伦勃朗光戏剧感），每场景至少2-3处暖光源 |
| **情绪关键词** | 安静、私密、温暖角落、猫咪的庇护所——安静但不阴森，治愈但不甜腻 |
| **角色比例** | 3-4头身（偏写实Q版融合，类似锈湖角色比例但更圆润） |
| **线条风格** | 有笔触质感的不规则手绘线条（非均匀粗细） |
| **纹理叠加** | 纸张纹理 + 轻微噪点 + 边缘做旧 |

## 二、场景美术资产（移步换景房间制）

### 2.1 场景架构说明

- 固定机位：每个房间使用一个精心构图的固定视角
- 点击切换：点击门/走廊/楼梯触发场景切换
- 过渡效果：淡入淡出/推拉/溶解，切换时短暂显示场景名称手写字
- 场景层级：背景层 → 家具层 → 角色层 → 光照后处理层
- 独立氛围：每个房间有独立的色调、光影和BGM参数
- 分辨率：场景背景2048px基准（支持2K显示），可等比放大至4K

### 2.2 场景清单

#### 2.2.1 主大厅（核心经营区）

| 属性 | 规格 |
|------|------|
| **数量** | 1幅全屏场景原画 + 昼夜变体（白天版/夜晚版） |
| **视角** | 略微俯视45°视角（非正交等距） |
| **构图要求** | 前景：吧台一角/木质桌面；中景：猫咪活动区（猫爬架/软垫/食盆）；远景：大窗户（可看到夜色街景）/通往其他房间的门 |
| **光源** | 天花板暖黄吊灯 + 窗户透入的月光/日光 |
| **氛围** | 老旧猫咖但正在被悉心经营，安静而温暖 |
| **分辨率** | 1920×1080基准（可等比放大至4K），图层分离交付 |
| **图层要求** | 背景远景 → 中景家具底座 → 前景遮挡 → 光照叠加层（共4-5层，供引擎分层控制） |
| **出口标记** | 通往静音隔间的门、通往后院花园的门、通往二楼阁楼的楼梯（每个出口需有微弱光从门缝透出/门框泛光的视觉引导） |
| **交付格式** | PSD分层源文件 + PNG透明底各层切片 |
| **优先级** | P0（原型验证阶段，W1-W4） |

**昼夜变体差异：**
- 白天版：窗外日光偏暖白 + 室内暖调；光线较均匀
- 夜晚版：窗外暗蓝夜空 + 室内暖灯对比强烈；局部光照聚焦更明显，阴影更深

## 十、猫咪动画详细规格补充

### 10.1 动画制作统一规范

| 维度 | 规格 |
|------|------|
| **帧率** | 12fps（手绘序列帧） |
| **每动画帧数** | 8-12帧/动画（核心动画12帧，次要动画8帧） |
| **动画总数** | 每只猫约15个意图对应动画（约120-180帧/猫） |
| **精灵尺寸** | 512-1024px（依猫咪体型而定） |
| **动画状态机层级** | BaseLayer（基础移动）+ ActionLayer（交互动作）+ EmotionLayer（表情叠加） |
| **动画过渡** | 需提供过渡关键帧，确保意图切换流畅不跳帧 |
| **交付格式** | PNG序列帧(透明底) + Unity SpriteSheet图集 + AnimatorController配置参考 |
| **SpriteResolver支持** | 部分动画可共用基础帧+SpriteLibrary管理变体以减少工作量 |

### 10.2 动画优先级分阶段策略

**P0阶段（原型验证）每只猫只做5个核心动画：**
1. Idle（待机呼吸）—— 最高优先级
2. Walk（移动）—— 最高优先级
3. Sleep（睡觉）—— 展示昼夜循环
4. Eat（进食）—— 展示基础交互
5. 1个最代表性格的特殊动作（奥利奥：哈气/小雪：躲藏/橘子：蹦跳）

**P1阶段补充剩余10个意图动画**
